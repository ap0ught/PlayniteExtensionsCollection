# Steam Game Importer

Extension to add games from steam with manual input of steam Id or store link and importing a Depressurized file export.

## Installation

### From Playnite Add-ons Browser
1. Open Playnite
2. Go to Add-ons (press F9)
3. Navigate to Browse tab
4. Search for "Steam Game Importer"
5. Click Install

### Manual Installation
1. Download the latest release from the [Playnite Add-ons Database](https://playnite.link/addons.html#Steam_Game_Importer)
2. Extract the downloaded file
3. Copy the extension folder to your Playnite extensions directory
4. Restart Playnite
