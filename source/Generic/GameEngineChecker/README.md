# Game Engine Checker

This extension will add a tag with the name of the engine used by the game. It's compatible with PC games that are available on GOG or Steam. The information is obtained from PCGamingWiki.

## Installation

### From Playnite Add-ons Browser
1. Open Playnite
2. Go to Add-ons (press F9)
3. Navigate to Browse tab
4. Search for "Game Engine Checker"
5. Click Install

### Manual Installation
1. Download the latest release from the [Playnite Add-ons Database](https://playnite.link/addons.html#Game_Engine_Checker_7a21243e-c7cc-4ca7-85bd-f6f96f22e9db)
2. Extract the downloaded file
3. Copy the extension folder to your Playnite extensions directory
4. Restart Playnite
