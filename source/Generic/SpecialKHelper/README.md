# Special K Helper

Provides functionality to improve using Special K with Playnite.

## Installation

### From Playnite Add-ons Browser
1. Open Playnite
2. Go to Add-ons (press F9)
3. Navigate to Browse tab
4. Search for "Special K Helper"
5. Click Install

### Manual Installation
1. Download the latest release from the [Playnite Add-ons Database](https://playnite.link/addons.html#SpecialKHelper_71349310-9ed8-4bf5-8bf2-e92cdb222748)
2. Extract the downloaded file
3. Copy the extension folder to your Playnite extensions directory
4. Restart Playnite
