# Steam Shortcuts

This extension adds convenient menu items to the right-click context menu of games, allowing to quickly access a variety of Steam game related pages and components.

## Installation

### From Playnite Add-ons Browser
1. Open Playnite
2. Go to Add-ons (press F9)
3. Navigate to Browse tab
4. Search for "Steam Shortcuts"
5. Click Install

### Manual Installation
1. Download the latest release from the [Playnite Add-ons Database](https://playnite.link/addons.html#Steam_Viewer_0a3edabb-065f-4056-a294-d6bc0656e2ac)
2. Extract the downloaded file
3. Copy the extension folder to your Playnite extensions directory
4. Restart Playnite
