# Steam Wishlist Discount Notifier

Checks your Steam wishlist in a view in Playnite and checks and notified you of new discounts in the background.

## Installation

### From Playnite Add-ons Browser
1. Open Playnite
2. Go to Add-ons (press F9)
3. Navigate to Browse tab
4. Search for "Steam Wishlist Discount Notifier"
5. Click Install

### Manual Installation
1. Download the latest release from the [Playnite Add-ons Database](https://playnite.link/addons.html#SteamWishlistDiscountNotifier_d5825a82-42bf-426b-ac47-5bea5df7aede)
2. Extract the downloaded file
3. Copy the extension folder to your Playnite extensions directory
4. Restart Playnite
