# Links Sorter

Simple extension that sorts the links of selected or all games in database by URL in ascending order (From A to Z, 0 to 9).

## Installation

### From Playnite Add-ons Browser
1. Open Playnite
2. Go to Add-ons (press F9)
3. Navigate to Browse tab
4. Search for "Links Sorter"
5. Click Install

### Manual Installation
1. Download the latest release from the [Playnite Add-ons Database](https://playnite.link/addons.html#Links_Sorter)
2. Extract the downloaded file
3. Copy the extension folder to your Playnite extensions directory
4. Restart Playnite
