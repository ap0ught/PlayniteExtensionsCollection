# Steam Launcher Utility

This extension allows you to configure the way steam is launched when starting a Steam game in both Playnite Desktop and Fullscreen mode.

## Installation

### From Playnite Add-ons Browser
1. Open Playnite
2. Go to Add-ons (press F9)
3. Navigate to Browse tab
4. Search for "Steam Launcher Utility"
5. Click Install

### Manual Installation
1. Download the latest release from the [Playnite Add-ons Database](https://playnite.link/addons.html#SteamLauncherUtility_31a65402-5b0c-44f0-9fc2-44b22ca4263c)
2. Extract the downloaded file
3. Copy the extension folder to your Playnite extensions directory
4. Restart Playnite
