# Importer for Anilist

This plugin imports your anime and manga lists from AniList for viewing in Playnite.

## Installation

### From Playnite Add-ons Browser
1. Open Playnite
2. Go to Add-ons (press F9)
3. Navigate to Browse tab
4. Search for "Importer for Anilist"
5. Click Install

### Manual Installation
1. Download the latest release from the [Playnite Add-ons Database](https://playnite.link/addons.html#ImporterforAnilist_2366fb38-bf25-45ea-9a78-dcc797ee83c3)
2. Extract the downloaded file
3. Copy the extension folder to your Playnite extensions directory
4. Restart Playnite
